# Frontend Style Guide

## Typography and Spacing Rules

1. Use typography tokens from `/frontend/src/styles/tokens.css` for all text sizing.
2. Do not use ad-hoc numeric `font-size` values in page or component styles.
3. Use spacing tokens for top/bottom margins and padding.
4. Avoid inline style spacing in TSX (`style={{ marginTop: ... }}` or similar); use class-based styles instead.
